# Contributing

Thanks for your interest! This repo is small and educational. If you'd like to contribute:

1. Fork the repo and create a branch.
2. Add changes (docs, small scripts, sanitized examples).
3. Do not add real IP addresses or API keys.
4. Open a pull request describing the change.

All contributions should follow the project's emphasis on ethical, authorized use only.
